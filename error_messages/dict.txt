Please specify name of file to save

bad example:
    my_dict = {
        'key1': 'val1',
        'key2': 'val2',
    }
    dump(my_dict)  # no name specified

first good example:
    my_dict = {
        'key1': 'val1',
        'key2': 'val2',
        'name': 'my_dict.yml',  # name specified here
    }
    dump(my_dict)

second good example:
    my_dict = {
        'key1': 'val1',
        'key2': 'val2',
    }
    dump(my_dict, 'my_dict.yml')  # name specified here
